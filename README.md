## ğŸš€ Quick Start
Follow the steps below to get the project humming locally.

```bash
# 1. Clone the repository
$ git clone https://github.com/YOUR_USERNAME/audio-player-react.git
$ cd audio-player-react

# 2. Install dependencies (requires Node.js 18 or newer)
$ npm install

# 3. Launch the development server with hot reload
$ npm start
```

Once the server is up, visit **http://localhost:3000** to explore the playlist.

## â“ FAQ
**Why Node.js 18 or later?**  
The toolchain and dependencies leverage modern Node features and ESM compatibility that ship with Node 18+.

**Can I use my own audio files?**  
Absolutely! Replace the `url` values in `home.js` with paths to your assets (local or remote) and update the `category`/`name` fields as needed.

**How do I deploy to production?**  
Run `npm run build` to generate the production bundle, then `npm run deploy:github` to push it live via GitHub Pages.
